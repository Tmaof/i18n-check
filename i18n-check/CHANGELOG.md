# Changelog

本文档记录了 i18n-check 项目的所有重要变更。

## [1.0.7] - 2025-12-06

- 解决 vue 文件没有 <script></script> 时， 在自动引入 i18n 时报错。

## [1.0.6] - 2025-12-06

- readme文档修改。

## [1.0.5] - 2025-12-05

### 修复

- 修复 HTML/Vue 模板中的注释文本内容被错误处理的问题，现在会忽略注释内容

## [1.0.4] - 2025-12-05

-
- 添加 `jsxChineseRegex` 参数，可以自定义 没有被引号包裹的中文字符的正则表达式
- 返回 `extractList`，对文件内容进行文本匹配提取后的结果列表

## [1.0.1] - 2025-11-30

### 功能

- 初始版本发布
- 支持自动检测代码中的中文文本
- 支持自动为中文文本添加 `i18n.t()` 包裹
- 支持自动导入 i18n 模块
- 支持智能忽略注释、枚举等不需要翻译的文本
- 支持模板字符串标记
- 支持提取翻译 Key
- 支持调用 LLM API 进行批量翻译处理
